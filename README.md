# Install
1. Download the code however you like  
2. Run `pip install -r requirements.txt`  
3. Run `python run.py`  
4. Copy-paste the numbers in the console
5. Wait a bit (it's a bit slow because of the API)
6. After it's done you can open index.html
7. Have fun! :D
